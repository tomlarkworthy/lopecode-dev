# Observable Keyboard Shortcuts

Source: https://observablehq.com/documentation/misc/keyboard-shortcuts

## Navigating between cells

| Shortcut | Action |
|----------|--------|
| Alt-Shift-Down | Focus next cell |
| Alt-Shift-Up | Focus previous cell |
| Mod-J | At cell reference, jump to definition |
| Shift-Ctrl-J | Jump to previously-focused cell |
| Ctrl-Shift-Down | Move cell down |
| Ctrl-Shift-Up | Move cell up |
| Escape | Blur cell |

## Inserting and deleting cells

| Shortcut | Action |
|----------|--------|
| Mod-Enter | Insert cell below |
| Shift-Ctrl-Enter | Insert cell above |
| Alt-Enter | Split cell, then focus below |
| Alt-Shift-Enter | Split cell, then focus above |
| Alt-Backspace | At cell start, merge with cell above |
| Ctrl-Alt-D | At cell end, merge with cell below |
| Alt-Shift-Cmd-Down | Copy cell and insert it below |
| Alt-Shift-Cmd-Up | Copy cell and insert it above |

| Shortcut | Action |

| Shift-Enter / Mod-S | Run cell |

| Ctrl-Shift-P / Mod-. | Pin or unpin editor |
| Mod-, | Show or hide output |
| Alt-Shift-F | Format code automatically |
| Shift-Tab | Indent automatically |
| Mod-] | Indent more |
| Mod-[ | Indent less |
| Mod-/ | Comment lines |
| Ctrl-Backspace | Delete to start of word before cursor |
| Enter | Insert new line and indent |

## Find and Replace

| Shortcut | Action |
|----------|--------|
| Shift-Mod-F | Open the Find pane |
| Mod-G | Find next occurrence |
| Shift-Mod-G | Find previous occurrence |
| Enter | In Find input, find next occurrence |
| Shift-Enter | In Find input, find previous occurrence |
| Enter | In Replace input, replace current occurrence |

| Mod-V | Paste clipboard contents |
| Ctrl-K | Kill text to end of line |
| Ctrl-Y | Yank previously killed text |
